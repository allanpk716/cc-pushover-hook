# Claude Code 开发规则

## 语言和交流

- 使用中文回答问题和交流

## 开发环境

- 当前的开发系统是 Windows
- 命令和脚本需考虑 Windows 兼容性

## BAT 脚本规范

- 开发或编辑 BAT 脚本时，**脚本内容不要包含中文字符**
- 仅在注释、输出消息或文档中使用中文
- 这是为了避免 Windows 控制台编码问题（CP936/GBK vs UTF-8）

## 脚本修改原则

- 修复脚本时，**优先在原有脚本上修改**
- 非必需情况不要新建脚本文件
- 这可以减少文件冗余，便于维护

---

*以上规则适用于此项目的所有开发工作*
